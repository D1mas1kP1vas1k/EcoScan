
  # 🌿 EcoScan — Твой персональный эко-ассистент

**EcoScan** — это мобильное приложение, которое превращает сложный состав продуктов в простую и понятную экологическую оценку. Мы помогаем делать осознанный выбор прямо у магазинной полки, сканируя штрихкоды товаров.

---

* **[🌐 Запустить в браузере (Web-версия)](https://D1mas1kP1vas1k.github.io/EcoScan)** — работает на смартфонах и ПК.

---

## ✨ Основные возможности

* 🔍 **Мгновенный сканер:** Распознавание товара по штрихкоду через камеру смартфона.
* 📊 **EcoScore (A-E):** Понятный рейтинг влияния продукта на окружающую среду.
* ♻️ **Анализ упаковки:** Проверка материалов на возможность переработки (пластик, стекло, бумага).
* 🌍 **Углеродный след:** Оценка объема выбросов CO₂ на этапе производства.
* 🧪 **Чистота состава:** Выявление вредных добавок и уровня промышленной обработки (NOVA).

---

## 🛠 Как это работает (Простыми словами)

Приложение выступает «умным фильтром» между мировыми базами данных и покупателем:

1.  **Запрос:** При сканировании приложение обращается к **OpenFoodFacts API** — глобальной энциклопедии продуктов.
2.  **Анализ:** Наш алгоритм обрабатывает «сырые» данные о составе и упаковке:
    * **Бонусы:** за органические сертификаты (Organic) и экологичную упаковку.
    * **Штрафы:** за наличие лишней химии и неперерабатываемого пластика.
3.  **Результат:** Вы получаете наглядную карточку с эко-рейтингом и советами по утилизации.

## 💻 Технический стек

* **Core:** React + TypeScript (быстрая и надежная логика).
* **Bundler:** Vite (мгновенная сборка).
* **Mobile:** Capacitor (превращение веб-кода в нативное Android-приложение).
* **API:** OpenFoodFacts (мировая база данных).
* **Hosting:** GitHub Pages.
